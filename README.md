# 문자 발송 시스템

## 프로젝트 목적

원하는 거래처에 한번에 문자를 보내기

## 사용 기술

    Spring boot  /  Spring Security

    JPA  /  Querydsl  /  PostgreSQL

    Apache POI  /  Thymeleaf  /  DataTables

    Bitbucket  /  EC2  /  RDS

## 주요 기능

* POI 라이브러리를 이용해서 업로드한 엑셀의 데이터를 자바 객체로 변환하기


* 검색 조건을 통해서 원하는 거래처 데이터 불러오기


* 원하는 거래처에 한번에 문자 발송하기
